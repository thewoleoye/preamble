General content of html: These are contents that give structure to

a webpage and organizes it

A self closing tag like line break <br/> should always have the slash to

indicate the closing

A comment <!--text--> is a special type of element that is used

to insert information for other developers to read

ELEMENTS CAN BE GROUPED INTO TWO:

1 BLOCK and 2 INLINE ELEMENT

Block element known as Division <div> will generally take up the entire

width of the screen whereas the inline element known as <span>

allows other inline elements to be displayed side by side with it

These elements are semantic elements which mean they don't

provide any inherent styling they are rather used to group like contents into

specified parts of the page; meaning content that should appear at the top of

the page goes in a header tag which is a different tag from head tag
